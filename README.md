# videomarker
我们看视频教程时，经常会遇到某个关键点需要记笔记，但是停下来会打断思路。看完再回去找可能又忘了在哪个位置了。这个插件可以标注视频中的“记忆点”，为这些记忆点添加备注和笔记。不同类型的视频可以被安排在不同文件夹中。每个视频的所有记忆点的笔记，可以合并导出一个markdown文档。

在全屏播放视频时，画面右下方会有一个钉子图案的悬浮图标。点击这个图标，视频将暂停。此时我们可以键入“备注”信息，这是对该记忆点的简短描述，按下回车，此记忆点就会被保存。不写备注也可以，直接再按一次钉子图案，就会创建一个无备注的记忆点。按下图标暂停视频后，若点击屏幕其他地方继续视频播放，则取消此次保存。

在浏览器任务栏打开插件页面。可以管理所有视频的记忆点体系。每个视频的记忆点归为一组，点击视频标题，可以展开显示该视频下的所有记忆点，在该记忆点撰写“笔记”，或删除该记忆点。点击“导出”，该视频笔记的各个记忆点的内容会被整合为一个markdown文件输出。点击“预览”，会跳转到一个markdown在线编辑器网页，我们可以查看当前md笔记的效果。

不同内容类型的视频，可以放置在不同的文件夹中。例如，当我们开启一个新的科目的学习时，可以创建一个新的文件夹。新创建了记忆点的视频笔记，会放在默认文件夹中，点击“移动”可以将该视频笔记移动到我们想要放置的文件夹。
